# Libros de cátedra

Este repositorio reúne los libros y materiales de las materias Desarrollo Web 1, Desarrollo Web II y Sistemas Paralelos, asignaturas dictadas por Federico Gonzalez en la Universidad Nacional de Tierra del Fuego (UNTDF), Argentina.

La propuesta está pensada como material de acompañamiento para la cursada, con un enfoque progresivo, práctico y orientado a quienes están aprendiendo a construir software y aplicaciones web desde sus fundamentos. En el caso de Desarrollo Web 1, el recorrido trabaja primero la estructura con HTML, después la presentación con CSS y finalmente la interacción con JavaScript, combinando explicaciones conceptuales, ejemplos y ejercicios con un proyecto transversal.

## Sobre el contenido

Los libros fueron elaborados como bibliografía y material de apoyo para estudiantes de la UNTDF. El objetivo es ofrecer una guía accesible y gradual para estudiar los temas centrales de cada materia, con explicaciones claras, práctica guiada y ejemplos que puedan reproducirse fácilmente con un editor y un navegador.

Actualmente, este repositorio organiza los contenidos en formato abierto y editable, para que puedan mantenerse, corregirse y ampliarse con facilidad.

## Tecnología utilizada

Todo el material está escrito en Markdown y se compila con Quarto. Eso permite trabajar los capítulos como archivos de texto simples, versionarlos con Git y generar libros o sitios estáticos de manera ordenada.

## Autoría

Federico Gonzalez Brizzio es el autor principal de estos materiales y jefe de cátedra de Desarrollo Web 1 en la UNTDF, donde además tiene a su cargo las asignaturas Laboratorio de Software, Sistemas Paralelos y Desarrollo Web II. El material fue pensado para acompañar procesos de aprendizaje iniciales y también como base de consulta para las cursadas.

En el libro de Desarrollo Web 1 también participaron Nicolás Sartori en la revisión de contenidos y Diego Abbá en la ilustración general y las infografías.
