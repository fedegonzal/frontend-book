## Parte 1: Fundamentos del desarrollo web moderno

- Introducción a JavaScript moderno
- Async y Await
- Uso de APIs REST
- Introducción a TypeScript (uso progresivo en React con JSX/TSX)
- Introducción a Git
- Node.js y npm

## Parte 2: Desarrollo basado en componentes

- Concepto de componentes
- SPA vs MPA
- Introducción a Web Components (estándar W3C)
- Introducción a Lit (visión general)
- Arquitectura de frontend:
  - Organización de carpetas
  - Componentes presentacionales vs contenedores
  - Separación de responsabilidades

---

## Parte 3: Desarrollo web con React

- Introducción a React
- JSX y TSX
- Componentes
- Props y State
- Renderizado condicional y listas
- Eventos
- Manejo de formularios con React Hook Form
- Hooks:
  - useState
  - useEffect
  - useContext
- Manejo de rutas con React Router
- Manejo de estado global con Zustand
- Consumo de APIs:
  - Fetching de datos
  - Manejo de estados: loading, error, success
- Librerías de componentes
- Testing:
  - Jest
  - React Testing Library
